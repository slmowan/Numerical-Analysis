#### Topics



**mit-lec29-奇异值分解：**

可以类比矩阵的相似对角化，只不过相似对角化只需要一个unitary阵而奇异值分解需要两个。

这个讲的挺好的，主要是理解奇异值分解的目标（在行空间中找到一组特殊的正交基，将其通过矩阵A线性变换后生成列空间中的另一组正交基），以及奇异值分解的本质（在矩阵的四个字空间中找到合适的basis）。
